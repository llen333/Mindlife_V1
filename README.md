# MindLife 🦞

Application de gestion de vie personnelle - Objectifs, Tâches, Calendrier, Habitudes, Journal.

## 🚀 Installation rapide

### Prérequis
- [Bun](https://bun.sh) installé sur votre machine

### Étapes

```bash
# 1. Aller dans le dossier
cd mindlife

# 2. Installer les dépendances
bun install

# 3. Initialiser la base de données et peupler avec les données de test
bun run setup

# 4. Lancer l'application
bun run dev
```

L'application sera disponible sur **http://localhost:3000**

## 📦 Scripts disponibles

| Commande | Description |
|----------|-------------|
| `bun run dev` | Lance le serveur de développement |
| `bun run setup` | Initialise la BDD + données de test |
| `bun run seed` | Repeuple la BDD avec les données de test |
| `bun run build` | Build de production |
| `bun run lint` | Vérifie le code |
