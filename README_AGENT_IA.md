# MindLife - Installation Locale Mac

## 🚀 Installation Rapide

### Prérequis
- **Bun** installé (https://bun.sh)
- **Node.js** 18+ (optionnel, Bun suffit)

### Installation

```bash
# 1. Décompresser le ZIP
unzip Mindlife_Agent_IA.zip
cd mindlife

# 2. Rendre le script exécutable
chmod +x setup-local.sh

# 3. Lancer l'installation
./setup-local.sh

# 4. Démarrer l'application
bun run dev
```

## 📍 Accès

L'application sera disponible sur: **http://localhost:3000**

## 🤖 Agent IA Intelligent

MindLife intègre maintenant un **Agent IA intelligent** avec plusieurs personas:

| Persona | Description |
|---------|-------------|
| 🪄 **Assistant** | Aide générale et organisation |
| 🏋️ **Coach Sport** | Entraînements et nutrition sportive |
| 🍎 **Nutrition** | Conseils nutritionnels et recettes |
| 🧠 **Productivité** | Gestion du temps et priorités |
| 💗 **Bien-être** | Équilibre vie pro/perso, méditation |

### Utilisation

1. Cliquez sur le bouton **✨** en bas à droite
2. Sélectionnez un persona dans le menu
3. Discutez avec l'IA en texte ou en voix
